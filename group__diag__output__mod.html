Parameters
[in]file_nameOutput file name
[in]file_titleDescriptive title for the file
[in]formatFile format (Currently only 'NETCDF' is valid)
[out]file_unitFile unit number assigned to the output file. Needed for subsuquent calls to diag_output_mod
[in]domainuThe unstructure domain

If the file name has .tileX or .tileX.nc where X is a one or two digit tile number, removes that suffix from the time name because fms2_io will add it

Note
If mpp_domains accepts more than 99 tiles, this will need to be updated

One-digit tile numbers example 

 filename.tile1.nc
       09876543210
          ^  ^
 filename.tile1
    09876543210
          ^  ^

Two-digit tile numbers example 

 filename.tile10.nc
        09876543210
          ^  ^
 filename.tile10
     09876543210
          ^  ^

If there is a .nc suffix on the file name, removes the .nc

Checks to make sure that only domain2D or domainUG is used. If both are not null, then FATAL
