19 !> @defgroup constants_mod constants_mod

21 !> @brief Defines useful constants for Earth. Constants are defined as real
22 !! parameters. Constants are accessed through the "use" statement.
23 !> @author Bruce Wyman
24 !!
25 !> Constants have been declared as type REAL, PARAMETER.
26 !!
27 !! The value a constant can not be changed in a users program.
28 !! New constants can be defined in terms of values from the
29 !! constants module using a parameter statement.<br><br>
30 !!
31 !! The name given to a particular constant may be changed.<br><br>
32 !!
33 !! Constants can be used on the right side on an assignment statement
34 !! (their value can not be reassigned).
35 !!
36 !! Example:
37 !!
38 !! @verbatim
39 !! use constants_mod, only: TFREEZE, grav_new => GRAV
40 !! real, parameter :: grav_inv = 1.0 / grav_new
41 !! tempc(:,:,:) = tempk(:,:,:) - TFREEZE
42 !! geopotential(:,:) = height(:,:) * grav_new
43 !! @endverbatim
